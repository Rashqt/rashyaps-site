RashYaps is more than just a blog.
It’s a mix of:

* 🧠 Thoughts & blogs

* 🖤 Dark aesthetic UI (minimal & smooth)
* ✍️ Blog system with posts
* ❤️ Like & 💬 comment system (JS-based)
* 🔗 Social media sharing (WhatsApp, X, LinkedIn, etc.)

* 📖 Featured project: *Uncrossed Paths*
* ☕ Support section (Buy me a coffee / UPI)
* 📩 Contact form
* 📱 Fully responsive design
* 🎬 Smooth animations & transitions
